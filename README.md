### The Hard Method
```js
GET QR
$ apt install git
$ apt install nodejs --fix-missing
$ git clone https://github.com/sajasxbot/MAALUTTY_V2
$ cd MAALUTTY_V2

$ npm install chalk
$ node maalutty.js
```
